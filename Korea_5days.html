每段交通/步行都标注了大致时长、导航关键字与说明，适合打印或离线保存。

行程概览

Day1–Day3：首尔（第3日晚乘KTX前往釜山） · Day4–Day5：釜山（第5日晚飞返上海）

Day 1 — 首尔（传统线）

时间段 行程 / 说明 交通 / 导航关键词 预计用时
上午 景福宫（Gyeongbokgung Palace）
参观宫殿、换韩服拍照（若租韩服，预留30–60分钟更舒适）。随后步行前往北村。 		步行 10–15 分钟
Gyeongbokgung → Bukchon Hanok Village		 约1.5–2小时（含拍照）
中午 三清洞 / 土俗村参鸡汤（Tosokchon Samgyetang）
传统韩餐，三清洞巷弄适合慢吃慢逛。 		步行 5–10 分钟
Tosokchon Samgyetang / Samcheong-dong		 约1–1.5小时
下午 仁寺洞（Insadong）
茶馆、手工艺店、传统纪念品。适合慢逛与拍照。 		步行 10–15 分钟 或 公交 1 站
Insadong-gil		 1.5小时
晚上 明洞夜市（Myeongdong）
补妆品/韩货/街边小吃（辣炒年糕、糖饼、炸鸡）。 		地铁：安国站 → 忠武路/明洞站（约20–30分钟）
Myeongdong Walking Street		 2小时+

Day 2 — 首尔（现代与夜景）

时间段 行程 / 说明 交通 / 导航关键词 预计用时
上午 弘大（Hongdae）
街头艺人、特色咖啡馆、年轻潮店。周末有路演与跳蚤市场。 		地铁2号线 明洞→弘大入口（约25分钟）
Hongdae Shopping Street		 2–3小时
中午 嘉年华式午餐（炸鸡、韩式拌饭）后前往梨花女子大学附近逛街。 步行或地铁 2–5 分钟
Ewha Womans University		 1.5–2小时
下午 狎鸥亭 / 林荫道（Garosu-gil）
特色小店+网红咖啡馆，适合慢逛与拍照。 		地铁换乘（耗时约25–30分钟）
Garosu-gil / Sinsa-dong		 1.5–2小时
晚上 南山塔（N Seoul Tower）
傍晚上塔看日落到夜景，可选缆车或步行上山（缆车更轻松）。 		地铁到忠武路站 → 步行/缆车上山（约15分钟缆车）
Namsan Seoul Tower		 1.5–2小时

Day 3 — 乐天世界塔 + 汉江放松，傍晚KTX前往釜山

时间段 行程 / 说明 交通 / 导航关键词 预计用时
上午 乐天世界塔（Lotte World Tower）
登楼观景、购物。建议提前预订 Seoul Sky 观景台票以节省排队时间。 		地铁至蚕室站（约30分钟）
Lotte World Tower / Seoul Sky		 2小时
下午 汉江公园（Yeouido Hangang Park）
可租自行车或买炸鸡+啤酒野餐，放松为晚上的长途高铁留体力。 		地铁约30分钟到汝矣岛站
Yeouido Hangang Park		 1.5–2小时
傍晚/晚上 首尔站 → KTX → 釜山站
建议选择17:00左右或更晚的车次（约2.5小时）。到达釜山后乘地铁或计程车前往海云台或西面入住。 		地铁到首尔站（约20分钟） + KTX（2.5小时）
Seoul Station → Busan Station (KTX)		 总计约3–4小时（含换乘）

Day 4 — 釜山海岸线（海云台 / 广安里 / 扎嘎其）

时间段 行程 / 说明 交通 / 导航关键词 预计用时
上午 海云台（Haeundae Beach）
早晨散步、拍照，附近有海云台市场可吃午餐。 		步行可达（若住海云台）
Haeundae Beach		 1.5–2小时
中午 / 下午 APEC 世峰楼（Nurimaru） → 冬柏岛海边步道
海边慢步拍照，风景佳。随后搭地铁前往广安里观桥灯光（夜景最佳）。 		步行 10–20 分钟至APEC
APEC Nurimaru / Dongbaekseom		 2–3小时
晚上 广安里（Gwangalli） 夜景 → 扎嘎其市场（Jagalchi）
广安里看桥灯，晚餐到扎嘎其吃海鲜，或返回南浦洞逛街。 		地铁或出租约25–40分钟（视起点）
Gwangalli Beach / Jagalchi Market		 2–3小时

Day 5 — 釜山文化点 + 返程（金海机场）

时间段 行程 / 说明 交通 / 导航关键词 预计用时
上午 甘川文化村（Gamcheon Culture Village）
彩色山城，徒步+拍照（多坡道，建议穿舒适鞋）。从西面出发搭地铁+短公交约25–40分钟。 		西面地铁 → 土城站 → 转公交（约25–40分钟）
Gamcheon Culture Village		 1.5–2小时
中午 松岛天空步道（Songdo Skywalk）
海上玻璃栈道，体验海景步道。午后返回西面/乐天百货购物。 		公交或地铁+步行（约25分钟）
Songdo Skywalk		 1–1.5小时
下午 / 晚上 最后购物后前往金海机场（Gimhae）办理登机，建议至少提前2小时到达机场（国际航班）。 西面地铁2号线 → 轻轨/机场线（约40–50分钟）
Gimhae International Airport / Busan		 约1–1.5小时（交通） + 安检时间
实用小贴士
  • 交通卡：购买并充值 T-money（地铁/公交/便利店通用）。
  • KTX：建议提前在官方网站或票务平台预订并选座（高峰期座位紧张）。
  • 导航：推荐使用 Naver Map 或 Kakao Map（可输入上方关键词直达）。
  • 网络：建议eSIM或随身Wi‑Fi，首尔与釜山信号良好。
  • 穿鞋：甘川村与南山、松岛步道有坡道，建议穿舒适的步行鞋。
  • 出发时间：KTX 车程约2.5小时，含换乘建议至少预留 3–4 小时段用于从市区到车站与候车。
